🔗 ConectaTrampo UDI

O ConectaTrampo UDI é um projeto web desenvolvido com o objetivo de facilitar a divulgação e visualização de oportunidades de trabalho e serviços na cidade de Uberlândia.

A proposta do sistema é criar uma plataforma simples e acessível onde pessoas possam encontrar oportunidades de forma rápida, contribuindo para a organização e conexão entre trabalhadores e serviços locais.

🚀 Tecnologias Utilizadas
HTML
CSS
JavaScript
📋 Funcionalidades
Exibição de oportunidades de trabalho ou serviços
Interface simples e organizada
Navegação entre páginas
Estrutura inicial para um sistema de divulgação de vagas
Projeto focado em aprendizado e prática de desenvolvimento web
